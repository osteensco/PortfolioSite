A classic game with some enhancements

This was my take on a simple game built with Javascript, CSS, and a little HTML. Pong is a game almost everyone is familiar with, but I wanted to add a little bit more excitement. The ball will drop additional effects that the player or cpu can pick up altering different aspects of the game.

Linked lists and recursion

Each drop mechanic adds additional complexity to the game, a great example of which is spawning additional balls. I use recursive functions that manipulate linked lists to handle scoring and removing extra balls after a goal or when the game ends.

CSS effects and animations

Every object in the game is created with CSS and HTML or DOM objects created via Javascript. I also took advantage of what can be done in CSS to polish the game through animations and other visual effects. I use pairs of keyframes and sub classes to allow for paddle color changes and the visual indicator for when the ball hits the paddle. This way I can mimic using the same animation on an object indefinently based on collision checks.
