Backend

This site was built with Python and the Django framework on the backend. I used templates that build off of each other in a way that makes the site easy to scale up with minimum backend changes. The site is hosted on Heroku, and uses a Heroku hosted Postgresql database. Static files are stored and served through an AWS S3 bucket. The database is relational, and is organized in a way to allow for implementing easy navigation to various sections throughout the website.

Frontend

The frontend is built off of Bootstrap 5. I utilize what Bootstrap has out of the box and customized certain classes. The navigation bar is a good example of this, I added a frosted glass effect over the whole pane and added a shadowbox to house the main navigation links.

In terms of page structure, I use a base template that contains my head, navigation, and footer sections. The home page and the templates for projects and technologies are extended from the base template and each technology page uses the basetech template. Because projects will have varying amounts of visual aids and content, they each have a customized html file which is extended from the baseproj template.
