An automated Dashboard

This dashboard is proof of concept excercise for the final stage of my Data Pipeline project. I am executing API calls to ingest rainfall data that lands in Google Bigquery. With the paid version of Tableau I would be able to connect directly to my Bigquery table, however, Tableau Public will only allows automated extracts from Google Sheets. Luckily, Google Sheets allows you to set up a live connection with Bigquery, so for this project Google Sheets is essentially used as a staging area for the backend of my Dashboard.
